Перечисление цветов камней на доске.

| Значение | Описание |

| `Color.White` | Белый камень |

## Классы

### `Position` [↑](#документация-go-engine-python-bindings)

Структура для представления координат на доске.

#### Конструкторы

| Сигнатура | Описание |
|-----------|----------|
| `Position()` | Создаёт позицию с координатами (0, 0) |
| `Position(x: int, y: int)` | Создаёт позицию с заданными координатами |

#### Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `x` | `int` | Координата по горизонтали (0-based) |
| `y` | `int` | Координата по вертикали (0-based) |

#### Методы

| Метод | Описание |
|-------|----------|
| `__eq__(other)` | Проверка равенства позиций |
| `__ne__(other)` | Проверка неравенства позиций |
| `__lt__(other)` | Сравнение позиций (для сортировки) |
| `__repr__()` | Строковое представление: `<Position(x, y)>` |

---

### `Group` [↑](#документация-go-engine-python-bindings)

Класс, представляющий группу связанных камней одного цвета.

#### Конструкторы [↑](#классы)

| Сигнатура | Описание |
|-----------|----------|
| `Group()` | Создаёт пустую группу без цвета |
| `Group(color: Color)` | Создаёт пустую группу заданного цвета |

#### Атрибуты [↑](#классы)

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `color` | `Color` | Цвет камней в группе |
| `stones` | `List[Position]` | Список позиций камней группы |
| `liberties` | `Set[Position]` | Множество позиций дамэ (дыхательных точек) группы |

#### Методы [↑](#классы)

##### `Group.contains()` [↑](#классы)

```python
contains(x: int, y: int) -> bool
```

Проверяет, содержит ли группа камень в указанной позиции.

**Параметры:**
- `x`, `y` — координаты камня

**Возвращает:** `True`, если камень есть в группе

---

##### `Group.add_stone()` [↑](#классы)

```python
add_stone(x: int, y: int) -> None
```

Добавляет камень в группу.

**Параметры:**
- `x`, `y` — координаты добавляемого камня

---

##### `Group.remove_stone()` [↑](#классы)

```python
remove_stone(x: int, y: int) -> None
```

Удаляет камень из группы.

**Параметры:**
- `x`, `y` — координаты удаляемого камня

---

##### `Group.empty()` [↑](#классы)

```python
empty() -> bool
```

Проверяет, пуста ли группа (нет камней).

**Возвращает:** `True`, если группа не содержит камней

### `Board` [↑](#документация-go-engine-python-bindings)

Класс игровой доски Го.

#### Конструктор [↑](#классы)

| Сигнатура | Описание |
|-----------|----------|
| `Board(size: int)` | Создаёт доску размером `size × size` (обычно 9, 13 или 19) |

#### Методы [↑](#классы)

##### `Board.get_board_array()` [↑](#классы)

```python
get_board_array() -> List[List[int]]
```

Возвращает текущее состояние доски в виде двумерного массива.

**Возвращает:** Массив размером `size × size`, где:
- `0` — пустая точка
- `1` — чёрный камень
- `2` — белый камень

---

##### `Board.get_size()` [↑](#классы)

```python
get_size() -> int
```

Возвращает размер доски.

---

##### `Board.get_color()` [↑](#классы)

```python
get_color(pos: Position) -> Color
get_color(x: int, y: int) -> Color
```

Возвращает цвет камня в указанной позиции.

**Параметры:**
- `pos` — позиция на доске, или
- `x`, `y` — координаты

**Возвращает:** `Color.Black`, `Color.White` или `Color.None`

---

##### `Board.add_stone()` [↑](#классы)

```python
add_stone(pos: Position, color: Color) -> bool
add_stone(x: int, y: int, color: Color) -> bool
```

Размещает камень на доску с проверкой правил.

**Параметры:**
- `pos` / `x`, `y` — позиция размещения
- `color` — цвет камня

**Возвращает:** `True` если ход выполнен, `False` если ход невозможен (занято, ко, самоубийство)

**Проверяет:**
- Занятость точки
- Правило ко (запрет повторения позиции)
- Правило самоубийства (запрет ходов без дамэ)

---

##### `Board.remove_stone()` [↑](#классы)

```python
remove_stone(pos: Position) -> bool
remove_stone(x: int, y: int) -> bool
```

Удаляет камень с доски и пересчитывает группы.

**Параметры:**
- `pos` / `x`, `y` — позиция удаляемого камня

**Возвращает:** `True` если камень удалён, `False` если точка была пуста

---

##### `Board.simple_print()` [↑](#классы)

```python
simple_print() -> None
```

Выводит текстовое представление доски в консоль.

Формат: `+` — пусто, `B` — чёрный, `W` — белый

---

##### `Board.find_group_index()` [↑](#классы)

```python
find_group_index(pos: Position) -> int
```

Находит индекс группы, содержащей камень в указанной позиции.

**Параметры:**
- `pos` — позиция камня

**Возвращает:** Индекс группы или `-1`, если камня нет

### `SGFGame` [↑](#документация-go-engine-python-bindings)

Класс для работы с форматом записи партий SGF (Smart Game Format).

#### Конструкторы [↑](#классы)

| Сигнатура | Описание |
|-----------|----------|
| `SGFGame()` | Создаёт игру с доской 9×9 по умолчанию |
| `SGFGame(size: int)` | Создаёт игру с доской указанного размера (минимум 9) |

#### Методы [↑](#классы)

##### `SGFGame.add_move()` [↑](#классы)

```python
add_move(move: Move) -> None
```

Добавляет ход в запись партии.

**Параметры:**
- `move` — объект `Move` с информацией о ходе

---

##### `SGFGame.set_player_names()` [↑](#классы)

```python
set_player_names(black: str, white: str) -> None
```

Устанавливает имена игроков.

**Параметры:**
- `black` — имя чёрного игрока
- `white` — имя белого игрока

---

##### `SGFGame.set_result()` [↑](#классы)

```python
set_result(result: str) -> None
```

Устанавливает результат партии.

**Параметры:**
- `result` — строка результата (например, `"B+R"`, `"W+5.5"`, `"Void"`)

---

##### `SGFGame.pos_to_sgf()` [↑](#классы)

```python
pos_to_sgf(pos: Position) -> str
```

Конвертирует координаты в SGF-формат.

**Параметры:**
- `pos` — позиция на доске

**Возвращает:** Строку из двух символов (например, `"dd"` для точки 3-3 на доске 19×19)

---

##### `SGFGame.generate_sgf()` [↑](#классы)

```python
generate_sgf() -> str
```

Генерирует полную SGF-строку партии.

**Возвращает:** Строку в формате SGF с заголовком и всеми ходами

---

##### `SGFGame.save_to_file()` [↑](#классы)

```python
save_to_file(filename: str) -> bool
```

Сохраняет SGF-запись в файл.

**Параметры:**
- `filename` — путь к файлу

**Возвращает:** `True` при успешном сохранении, `False` при ошибке

---

##### `SGFGame.get_moves()` [↑](#классы)

```python
get_moves() -> List[Move]
```

Возвращает список всех ходов партии.

---

##### `SGFGame.clear()` [↑](#классы)

```python
clear() -> None
```

Очищает список ходов.

### `Game` [↑](#документация-go-engine-python-bindings)

Основной класс для управления игрой Го.

#### Конструктор [↑](#классы)

| Сигнатура | Описание |
|-----------|----------|
| `Game(size: int = 9)` | Создаёт игру с доской указанного размера (по умолчанию 9×9) |

#### Методы [↑](#классы)

##### `Game.is_ok()` [↑](#классы)

```python
is_ok(pos: Position, board: Board, player_color: Color) -> bool
```

Проверяет, является ли ход допустимым.

**Параметры:**
- `pos` — проверяемая позиция
- `board` — доска для проверки
- `player_color` — цвет игрока, делающего ход

**Проверяет:**
- Точка свободна
- Не нарушает правило ко
- Не является самоубийством

**Возвращает:** `True` если ход допустим

---

##### `Game.re_pos_moves()` [↑](#классы)

```python
re_pos_moves(rele_board: Board, player_color: Color) -> Board
```

Вычисляет доску возможных ходов.

**Параметры:**
- `rele_board` — текущая доска
- `player_color` — цвет игрока

**Возвращает:** `Board` где `Color.Black` помечает допустимые ходы

---

##### `Game.get_legal_moves()` [↑](#классы)

```python
get_legal_moves() -> Board
```

Возвращает доску с отмеченными допустимыми ходами для текущего игрока.

---

##### `Game.record_move()` [↑](#классы)

```python
record_move(x: int, y: int, is_pass: bool = False) -> None
```

Записывает ход в историю и SGF.

**Параметры:**
- `x`, `y` — координаты (или -1, -1 для паса)
- `is_pass` — `True` если ход — пас

---

##### `Game.undo_last_move()` [↑](#классы)

```python
undo_last_move() -> bool
```

Отменяет последний ход.

**Возвращает:** `True` при успехе, `False` если нет ходов для отмены

---

##### `Game.get_sgf()` [↑](#классы)

```python
get_sgf() -> str
```

Возвращает текущую SGF-запись партии.

---

##### `Game.save_game()` [↑](#классы)

```python
save_game(filepath: str) -> bool
```

Сохраняет игру в SGF файл.

**Параметры:**
- `filepath` — путь к файлу (создаёт директории при необходимости)

**Возвращает:** `True` при успешном сохранении

---

##### `Game.make_move()` [↑](#классы)

```python
make_move(x: int, y: int, is_pass: bool = False) -> bool
```

Выполняет ход или пас.

**Параметры:**
- `x`, `y` — координаты (игнорируются при `is_pass=True`)
- `is_pass` — `True` для паса

**Возвращает:** `True` если ход принят, `False` если отклонён

**Особенности:**
- При двух подряд пасах устанавливает `gameOver = True`
- Автоматически переключает текущего игрока
- Обновляет список допустимых ходов

---

##### `Game.is_game_over()` [↑](#классы)

```python
is_game_over() -> bool
```

Проверяет, закончена ли игра.

**Возвращает:** `True` после двух подряд пасов

---

##### `Game.get_current_player()` [↑](#классы)

```python
get_current_player() -> Color
```

Возвращает цвет текущего игрока.

---

##### `Game.get_move_number()` [↑](#классы)

```python
get_move_number() -> int
```

Возвращает номер текущего хода (начиная с 1).

---

##### `Game.get_passes()` [↑](#классы)

```python
get_passes() -> int
```

Возвращает количество подряд пасов.

---

## Примеры использования [↑](#документация-go-engine-python-bindings)

```python
import go_engine

# Создание игры 9x9
game = go_engine.Game(9)

# Выполнение ходов
game.make_move(3, 3)      # Чёрные на 3-3
game.make_move(4, 3)      # Белые
game.make_move(3, 4)      # Чёрные

# Проверка допустимых ходов
legal = game.get_legal_moves()
board_state = game.get_board().get_board_array()

# Получение SGF
sgf_string = game.get_sgf()

# Сохранение в файл
game.save_game("game.sgf")

# Пас и завершение игры
game.make_move(-1, -1, is_pass=True)
game.make_move(-1, -1, is_pass=True)  # Игра закончена

# Отмена хода
game.undo_last_move()
```
